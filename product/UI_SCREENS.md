### Component Breakdown

**Top Navigation Bar:**
- Logo (left, clickable â†’ home)
- Main navigation links (Dashboard, Calls, AI, Team, Analytics)
- Notification bell with badge count
- User avatar with dropdown menu

**Stats Cards (4x):**
- Icon (48x48px)
- Label (text-sm, gray-600)
- Value (text-h2, bold, gray-900)
- Change indicator (text-sm, with arrow, green/red)

**Live Calls Widget:**
- Header with count and "View All" link
- List of active calls (max 5 shown)
- Each row: phone number, language, status, duration, action button

**Charts:**
- Call Volume: Bar chart, 7 days, hover shows exact numbers
- Peak Hours: Horizontal bar chart showing busiest times

**Recent Activity:**
- Timeline list
- Icon + description + timestamp
- Max 5 items, "View All" link

**Quick Actions:**
- 5 most common actions
- Icon + label buttons
- Opens relevant page or modal

### Responsive Behavior

**Desktop (>1280px):**
- 4 stat cards in row
- 2-column layout for charts/activity

**Tablet (768px - 1280px):**
- 2 stat cards per row (2 rows)
- Single column for charts
- Sidebar collapses to icons

**Mobile (<768px):**
- 1 stat card per row (scrollable)
- Single column layout
- Sidebar hidden, hamburger menu
